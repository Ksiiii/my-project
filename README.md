# 科研背景：螺栓拧紧工艺智能助手（VDI2230 × 数据驱动建模）

本项目面向工程制造场景中的螺栓连接设计与拧紧工艺优化，目标是构建一个“螺栓拧紧工艺智能助手”，能够在给定螺栓/法兰几何、材料与载荷等条件下，快速、稳定地预测关键工艺与结构响应量，并为工程人员提供可解释、可落地的决策支持。

螺栓预紧力及其相关刚度/载荷分配参数是螺栓连接可靠性的核心，直接影响密封性能、疲劳寿命与失效风险。传统工程设计通常依赖标准化解析/半解析计算与经验规则，例如 VDI2230 体系能够在明确假设下给出一套可计算的基线结果。但在真实装配与服役过程中，几何公差、接触条件、材料批次差异、装配扰动与测量噪声等因素会造成与理想化模型存在偏差，使得仅依赖解析公式难以完全覆盖复杂工况与不确定性。

因此，本项目以 VDI2230 的物理基线为基础，结合扰动表格数据与数据驱动模型进行学习与修正：一方面利用标准公式提供可解释的先验结构，另一方面通过机器学习捕捉解析模型未显式建模的残差与分布变化，从而在保持工程一致性的前提下提升预测精度与稳健性。当前阶段的工作重点是建立可复现的基线实验框架，对比梯度提升树（GBDT）与基于 Transformer 的表格建模方法（FT-Transformer）在不同预测目标设置下的表现，并为后续进一步研究（如物理约束残差学习、分布外 OOD 泛化评估、与大模型 Agent 的集成）奠定可验证的实验基础与评价体系。
